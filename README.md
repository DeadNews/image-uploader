# py-image-uploader

Upload images via API

- Works out of the box:
  - fastpic
  - geekpic
  - pixhost

- Key required:
  - freeimage
  - imageban
  - imageshack
  - imgbb
  - imgur

## Installation

```sh
pip install py-image-uploader
```

## .env variables

Don't forget to specify [env variables](.env.sample).
